# Dark World — D20 Pack

Как использовать:
1) Скопируй *содержимое* этой папки в корень репозитория `dark-world` (папки `api`, `public`, `src`).
2) Замени `src/App.tsx` — это демо броска.
3) Свои 6 PNG-спрайтов положи в `public/assets/d20/` с теми же именами.
4) Локально `npm run dev` — на главной будет кнопка «Бросить кости».
5) Запушь изменения — Vercel соберёт проект. Edge-функция `/api/roll.ts` готова.
