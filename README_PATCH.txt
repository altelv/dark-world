# Patch: Vite alias resolution

Эта заплатка чинит ошибку Vite:
[vite]: Rollup failed to resolve import "@store/game" ...

Что входит:
- обновлённый `package.json` (добавлен плагин `vite-tsconfig-paths`)
- обновлённый `vite.config.ts` (подключён плагин)

Шаги:
1) Замените эти два файла в корне проекта.
2) Выполните установку зависимостей локально или дайте Vercel собрать сам:
   - локально: `npm i` затем `npm run build`
   - на Vercel: просто Redeploy — он сам установит новый плагин.
3) Проверьте, что билд проходит и приложение открывается.

Если хотите без плагина — можно вместо этого прописать вручную `resolve.alias` для всех путей из tsconfig.
